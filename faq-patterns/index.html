Tiled Layout & Layers Function

What is a tiled pattern layout?

A tiled pattern layout refers to a method of presenting a sewing pattern in a format that you can print on standard home printers. The pattern is divided into multiple pages in a grid-like arrangement, with each page containing a portion of the pattern. Once done with printing, you can trim and tape these pages together to create a full-size pattern. Each tile contains special triangular notches that indicate which tile you should tape where.

What is a layer function in the pattern files? How do I access it?

The layer function in pattern files allows you to selectively display or hide specific sizes or pattern elements within the digital pattern file. Thus, you can customize the pattern file‚Äôs display to suit your needs, making it easier to work with and reducing visual clutter.

To access and turn off layers, open the pattern file on your computer‚Ää* using the most current version of Adobe Acrobat Reader. Then, go to the drop-down menu View > Show > Navigation > Layers. Alternatively, you can press the F4 key, and the icons for Page, Thumbnails, Attachments, and Layers will appear on the side. Then, click the eye icon next to the sizes you don‚Äôt need to make them disappear.

*Unfortunately, as of the time this FAQ page is published, mobile apps for viewing and editing PDF files do not support the layers panel. You can view the instructions and attachments on your mobile device, but the layers function will be unavailable.
